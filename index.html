മലയാളത്തിൽ എളുപ്പത്തിൽ പഠിക്കാൻ!

React

React JS മുഴുവൻ ട്യൂട്ടോറിയൽ മലയാളത്തിൽ. Hooks, Routers, Redux എല്ലാം ഈ ട്യൂട്ടോറിയലിലൂടെ പൂർണമായും മനസ്സിലാക്കാൻ സാധിക്കും

Advanced Javascript

അത്യാവശ്യമായി അറിഞ്ഞിരിക്കേണ്ട ജാവാസ്ക്രിപ്റ്റിലെ എല്ലാ പുതിയ കോൺസെപ്റ്റുകൾ മലയാളത്തിൽ

Git

Git ഏതൊരു പ്രോഗ്രാമിങ്ങിനും അത്യാവശ്യം അറിഞ്ഞിരിക്കേണ്ട ഒന്നാണ്. Gitന്റെ പ്രധാനപ്പെട്ട എല്ലാ കോൺസെപ്റ്റുകളും ഇവിടെ പറയുന്നു.

Redux

Reactന്റെ ഒപ്പം ഉപയോഗിക്കുന്ന ഏറ്റുവും പ്രശസ്തമായ data management library ആണ് Redux. Reduxഉം ഒപ്പം Redux Toolkitഉം ഈ ട്യൂട്ടോറിയലിൽ മനസ്സിലാക്കാൻ സാധിക്കും.

React Router

Single Page application ആയ Reactൽ, URL routing സാധ്യമാകുന്ന library ആണ് React Router. വളരെ basic concept മുതൽ ഈ ട്യൂട്ടോറിയലിൽ പറയുന്നു
